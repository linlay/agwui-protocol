SSE Event Architecture

这版按功能邻近性把颜色收敛成 5 个主色族：蓝色族对应 request/chat/run， 青绿色族对应 reasoning/content，暖琥珀族对应 tool/action/awaiting， 琥珀偏紫强调色对应 artifact，灰紫族对应 plan/task。 树形结构只保留层级，具体事件的 #RRGGBB 只在右侧色卡展示。
